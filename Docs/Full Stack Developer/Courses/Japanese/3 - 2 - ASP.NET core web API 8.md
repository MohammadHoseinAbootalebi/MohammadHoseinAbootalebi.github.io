## 説明

現代のWebアプリケーションは、その機能の多くをAPIで提供しています。このコースでは、Webのパイオニアであり、テクノロジースペシャリスト、そして起業家でもあるChristian Wenzが、ASP.NET Core 8を使用してそのようなAPIを設計および実装する方法を紹介します。最小限のAPIと、HTTP、REST、そしてAPI設計の基礎を学びましょう。データの取得と書き込みの方法を学び、ハンズオンチャレンジで学んだことを実践してみましょう。
