## 描述

现代 Web 应用程序的大部分功能都通过 API 提供。在本课程中，Web 先驱、技术专家和企业家 Christian Wenz 将向您展示如何使用 ASP.NET Core 8 设计和实现这样的 API。从最精简的 API 开始，并学习 HTTP、REST 和 API 设计的基础知识。学习如何检索和写入数据，并通过实践挑战练习所学知识。
